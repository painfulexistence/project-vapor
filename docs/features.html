Features

Rendering

FeatureDescription
Forward + Tiled Light CullingPrimary render path. View frustum divided into a 16×16×24 cluster grid; each cluster computes only its affecting lights.
GPU-Driven Instancing5000+ simultaneous instances. Tested against the Bistro scene (2911 instances).
Physically-Based RenderingDisney BRDF shading model. Supports albedo, metallic, roughness, normal, and emissive maps.
MSAA 4xHardware multi-sample anti-aliasing.
Raytraced Hard ShadowsMetal only. Uses Metal ray tracing API for accurate hard shadows.
BloomHDR bloom post-processing.
Depth of FieldCoC → blur → composite pass infrastructure exists. Currently disabled (passes commented out).
Volumetric Fog / God RaysVolumetric fog and Tyndall light beam effect.
Volumetric CloudsProcedural volumetric cloud rendering.
Atmosphere RenderingRayleigh + Mie scattering atmosphere model.
Sun / Lens FlareSun lens flare effect.
Water RenderingNormal-map water surface with foam. Currently disabled (pass commented out).
2D Batch RenderingScreen-space quads, lines, rectangles, circles, triangles — batched.
3D Batch RenderingWorld-space batch rendering with depth testing.
Sprite / Texture QuadTextured quad rendering with rotation and tint.
Bitmap Font Rendering2D screen-space and 3D world-space text.
Multi-backendMetal (macOS) and Vulkan (cross-platform) sharing the same API surface.

Physics (Physics3D)

FeatureDescription
Rigid Body CreationSphere, box, capsule, cylinder, mesh, convex hull shapes.
Body LifecycleaddBody / removeBody / destroyBody.
Motion TypesStatic, Dynamic, Kinematic.
Force & ImpulseapplyForce, applyImpulse, applyTorque.
Velocity ControlsetLinearVelocity, setAngularVelocity.
Physical PropertiesMass, friction, restitution, linear/angular damping, gravity factor.
Trigger VolumesBox, sphere, capsule sensors with onTriggerEnter/Exit callbacks.
Overlap TestsoverlapSphere, overlapBox, overlapCapsule — returns overlapping nodes.
Raycastingraycast(from, to, hit, ignoreBody) with hit point, normal, distance, node pointer.
Physics Debug DrawWireframe overlay for collision geometry visualization.
Fixed-step Simulation60Hz fixed timestep with render interpolation alpha.
Shape CacheSame-dimension shapes are shared automatically.
Task-based ParallelismJolt runs on JPH::JobSystemThreadPool.
Character ControllerCapsule character controller handling steps, slopes, gravity.
Vehicle ControllerVehicle physics with axles and suspension.

Audio (AudioManager)

FeatureDescription
2D Audioplay2d(filename, loop, volume) — non-spatialized.
3D Spatial Audioplay3d(filename, position/config) — positional.
Playback Controlstop, pause, resume (single or all).
Audio PropertiesVolume, pitch, loop, seek, duration.
Distance ModelsNone, Linear, Inverse, Exponential.
Directional Audio ConesInner/outer angle + outer gain for directional sources.
Doppler EffectVia velocity3D.
Listener ControlPosition, velocity, orientation — wired to camera/player.
Completion CallbackFires on main thread when audio finishes.
Instance PoolUp to 32 simultaneous playback instances.
