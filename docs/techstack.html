Tech Stack

Third-party libraries, backend choices, and rationale.

Core Engine

LibraryRoleStatus
SDL3 Window management, input events, timing. Cross-platform surface for both Metal and Vulkan. Live
glm Math library — vec2/3/4, mat4, quat. Used throughout engine and demo. Live
EnTT ECS framework used in the Vaporware demo layer for game-specific components and systems. Live
enkiTS Task scheduler backing ResourceManager async loads. Live
fmt String formatting. Used for logging and debug output. Live
Tracy Frame profiler. CPU/GPU timing markers throughout the engine. Live

Graphics

BackendNotesStatus
Metal (via metal-cpp) Primary backend. macOS only. Full feature set including ray tracing, 31 render passes, all post-processing effects. Live
Vulkan (via glslang) Cross-platform backend. Currently 3 passes vs Metal's 31. Second-class status — batch rendering, fonts, UI, and volumetrics are missing. Live (partial)
DX12 Not a priority. Real-time ray tracing and mesh shading are the eventual motivation, but no timeline. Not started

Web platform is not supported and not planned.

Physics

LibraryRationaleStatus
Jolt Physics Chosen for multi-threading support. Jobs run on JPH::JobSystemThreadPool (migrated from enkiTS integration in Phase 4). Supports rigid bodies, character controller, vehicle physics, triggers, raycasting. Live

Audio

LibraryRationaleStatus
miniaudio Single-header audio engine. Handles 2D playback and 3D spatial audio with distance models, Doppler, and directional cones. No dependency overhead. Live

UI

LibraryRoleStatus
RmlUi In-game HTML/CSS UI framework. Used for HUD, subtitles, chapter titles, letterbox, scroll text. Documents loaded as .rml files. Live
ImGui Developer debug UI. Hosts the scene inspector with per-component custom drawers. Not shipped in release builds. Live

Asset Loading

LibraryRoleStatus
tinygltf GLTF 2.0 scene loading. Primary 3D asset format. Live
stb_image PNG/JPG/HDR image decoding. Live
MikkTSpace Mikktspace tangent space generation for normal mapping. Live
OpenUSD Intended for DCC interoperability. Not yet integrated. Planned
FlatBuffers Data serialization for engine assets. Not yet integrated. Planned

Scripting

OptionNotesStatus
Terra / Ravi JIT-capable Lua variants for native platforms. No JIT restrictions on desktop. No timeline yet. Planned

Editor

OptionNotesStatus
Avalonia UI or desktop-web stack Under consideration for a standalone editor. No decision made. Planned

Build & Infrastructure

ToolRoleStatus
CMake 3.24+ Build system. Presets for dev and release configurations. Live
Ninja Build backend. Required for CMake presets. Live
vcpkg Dependency management via vcpkg.json. All third-party libraries declared here. Live
ccache Compiler cache. Speeds up CI and local incremental builds. Live
Catch2 v3 Test framework. 76 tests across 5 binaries. Live
GitHub Actions CI on every push. Runner: macos-15, Ninja, ccache. Live
